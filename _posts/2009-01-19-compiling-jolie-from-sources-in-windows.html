--- layout: post title: Compiling JOLIE from sources in Windows now easier date: '2009-01-19T15:33:00.006+01:00' author: Fabrizio Montesi tags: - jolie modified_time: '2009-01-20T16:30:59.803+01:00' blogger_id: tag:blogger.com,1999:blog-3856624862131672592.post-3391305478933107237 blogger_orig_url: http://fmontesi.blogspot.com/2009/01/compiling-jolie-from-sources-in-windows.html --- For everyone using Windows out there: compiling JOLIE from sources in Windows is not a pain anymore!

I've patched the build system so that it recognizes the running operating system and installs the appropriate launcher script (a shell script in *nix and a .bat script in Windows), so now having a working environment is far less "manual" than before. You can find the updated guide here (Windows -> From sources).
